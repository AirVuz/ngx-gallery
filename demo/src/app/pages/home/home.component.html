Angular Image Gallery

An angular image gallery module that simplifies the process of creating beautiful image galleries for the web and mobile devices.

Features:

  • Thumbnails
  • Dots (bullets)
  • Text (image description)
  • Navigation
  • Slide show
  • Custom loader
  • Lazy load
  • Auto detect images
  • Responsive layout
  • Modal (lightbox)
  • Gestures support
