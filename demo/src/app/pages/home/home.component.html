Angular Image Gallery

An angular image gallery module that simplifies the process of creating beautiful image galleries for the web and mobile devices.

Easy customization, enable/disable features, set style and position to the gallery parts using gallery options
Responsive layout, works on mobile, tablet and desktop browsers.
Built to integrate seamlessly with Angular.

Features:

  • Gallery in modal
  • Dots (bullets)
  • Text (description)
  • Thumbnails
  • Lazy load
  • Navigation
  • Loading icons
  • Auto detect images
  • Slide show
  • Key shortcuts for navigation (gallery modal)
  • Responsive
  • Gestures support (hammerjs)
