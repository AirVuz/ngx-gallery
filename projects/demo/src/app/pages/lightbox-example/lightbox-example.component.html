Lightbox

Overview

Lightbox is a service that opens the gallery in an overlay.

Usage

First we need to display the thumbnails in the template, then we add a click event to open the lightbox.

Example

Lightbox config can be set globally in LightboxModule.forRoot(config) or directly with the open function.

By default, the lightbox uses full screen style on small screens like on mobile phones, but you can force a full screen style on large screens too.

For example, open the lightbox in mode

List of lightbox options is available in the Lightbox API
