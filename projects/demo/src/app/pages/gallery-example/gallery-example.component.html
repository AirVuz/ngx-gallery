Gallery

Overview

a slideshow component for cycling through elements - images, videos or iframes - like a carousel.

Usage

The most basic usage is to load items using an array of type GalleryItem[]

This is done using the template binding

Another way is to convert a stream of data to a stream of gallery items using the map operator.

This can be done using the async pipe to subscribe to the stream in the template.

Alternatively, The gallery can be accessed using the component reference.

Get the component using @ViewChild(GalleryComponent) then add items

Access the gallery from anywhere in your app.

Use the Gallery service to get the GalleryRef by id

Example

The gallery has many options that you can adjust, check all available options in Gallery API

Beside these options, you can also use CSS to change gallery's style.

For more advanced usage, check the Advanced Example
